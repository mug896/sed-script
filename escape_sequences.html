Escape sequences

Escape Sequence Character represented
\a alert (bell)
\f form feed
\n newline
\r carriage return
\t horizontal tab
\v vertical tab
\cx Ctrl-x 문자, 가령 \cz 는 Ctrl-z (^Z)
\dNNN 10 진수 ASCII 문자 NNN
\oNNN 8 진수 ASCII 문자 NNN
\xHH 16 진수 ASCII 문자 HH

\b (backspace) 는 regex 에서 word boundary (\b) 와 겹치므로 제외되었습니다.

# NUL 문자는 `\x0` 를 사용하면 됩니다.
$ find ...  -print0 | sed -z 'N; s/\x0/.../'

# 유니코드 문자는 다음과 같이 shell 의 $' ' 을 사용하면 됩니다.
$ echo aaa | sed 's/a/'$'\U2665''/g'
